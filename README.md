Smart Calculator 🧮

A Python-based Smart Calculator built using core Python and NumPy.
This project performs basic arithmetic operations along with NumPy statistical calculations through a simple menu-driven interface.

🚀 Features
Addition of multiple numbers
Subtraction
Multiplication of multiple numbers
Division
Power calculation
Modulus operation
NumPy Array Statistics:
Sum
Mean
Median
Standard Deviation
Variance
User-friendly menu system
Input validation for better reliability


🛠️ Technologies Used
Python 3
NumPy Library
📂 Project Structure
smart-calculator/
│
├── calculator.py
└── README.md

📦 Installation
Clone the repository:
git clone https://github.com/Bhavayami/smart-calculator.git
Navigate to the project folder:
cd smart-calculator
Install NumPy:
pip install numpy


▶️ How to Run

Run the Python file:

python calculator.py
📋 Menu Options
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Power
6. Modulus
7. NumPy Array Operations
8. Exit
